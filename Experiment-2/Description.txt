CSS STYLING BREAKDOWN

1. ELEMENT: body (Page Background)  
   Purpose: Sets the background and centers the content  
   Properties:  
   background-color: #f2f2f2;  
   text-align: center;  
   padding-top: 50px;  
   font-family: Arial, sans-serif;

2. ELEMENT: .bank-box (Main Box)  
   Purpose: Container that holds balance and buttons  
   Properties:  
   background-color: white;  
   padding: 20px;  
   width: 250px;  
   margin: auto;  
   border: 1px solid #ccc;

3. ELEMENT: .balance (Balance Text)  
   Purpose: Displays account balance  
   Properties:  
   font-size: 18px;  
   margin-bottom: 20px;

4. ELEMENT: .btn (Common Button Style)  
   Purpose: Base style for both buttons  
   Properties:  
   padding: 10px 15px;  
   font-size: 16px;  
   margin: 5px;

5. ELEMENT: .deposit-btn (Deposit Button)  
   Purpose: Applies green styling to deposit button  
   Properties:  
   background-color: green;
